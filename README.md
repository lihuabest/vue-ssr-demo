# vue-webpack-demo

运行步骤

第一步，安装依赖
```sh
  npm install
```

第二步，本地开发运行

```sh
  npm run start
```

第三步，代码打包

```sh
  npm run build
```

第三步，服务端运行

```sh
  npm run server
```
